# Python Vending Machine

Simple Python vending machine

## Installation

Download here
https://github.com/Chilia726/Python-Vending-Machine/releases/tag/3.0.1

```bash
pip install -r requirements.txt
```

## Usage

To run, do

```bash
py main.py
```
